Toestemmingen beheren ()
Het vermogen om toestemming tot toegang tot (markt) data te beheren. De toestemming wordt gebruikt bij het delen van data waarvoor toegang is vereist. Dit zijn onder meer de toestemmingen van klanten. Dit betreft een GUE verantwoordelijkheid.
